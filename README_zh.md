# Markdown 教程

Markdown 中文教程，完整指南。

[查看在线教程](https://albert-lsk.github.io/Markdown-Tutorial/)

## 特性

-   :art: **Claude 设计风格** — 采用 Anthropic 官方设计系统
-   🌙 **深色模式** — 自动跟随系统或手动切换
-   🇨🇳🇺🇸 **中英双语** — 随时切换语言
-   💬 **评论系统** — 基于 GitHub Discussions 的 Giscus 评论

## 技术栈

-   **框架**: [Astro](https://astro.build)
-   **样式**: [Tailwind CSS](https://tailwindcss.com)
-   **设计**: Claude (Anthropic)

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── pages/
│   ├── index.astro          # 中文首页
│   ├── en/                  # 英文页面
│   ├── basic/               # 基础语法
│   └── extended/            # 扩展语法
├── components/              # 组件
├── layouts/                 # 布局
├── styles/                  # 样式
└── i18n/                    # 国际化
```
